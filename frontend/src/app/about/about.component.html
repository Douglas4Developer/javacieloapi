Sobre o Projeto

Projeto de Integração de Pagamentos

Este projeto foi desenvolvido para gerenciar vendas e processar pagamentos de maneira segura e eficiente, utilizando a API da Cielo para transações de cartão de crédito. Implementamos funcionalidades completas de criação, listagem e cancelamento de vendas, com suporte a diferentes bandeiras de cartão, como Visa, Mastercard, American Express e Elo.

A aplicação oferece uma interface de usuário moderna, desenvolvida com Angular e Material Design, garantindo uma experiência fluida e intuitiva para os usuários. Além disso, o backend é construído utilizando Spring Boot, assegurando um alto nível de segurança e criptografia de dados sensíveis, como números de cartão e códigos de segurança.

Também incorporamos a geração de relatórios detalhados de vendas, com possibilidade de cancelar transações de forma rápida e eficiente, utilizando as melhores práticas de arquitetura de software e integração de APIs.
